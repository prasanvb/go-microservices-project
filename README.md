# go-microservices

## Frontend

- Navigate to Broker service
  - `cd frontend`
- Run the frontend app
  - `go run ./src/web/main.go`

## Broker service

- Navigate to Broker service
  - `cd broker-service`
- Run the broker service
  - `go run .src/api/main.go .src/api/routes.go .src/api/handlers.go`
